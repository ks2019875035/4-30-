안혜지의 가상컴퓨터

목적

최근들어 유튜브를 자주 접하면서 영상 편집하는것에 관심이 끌리게 되었습니다.
나중에 취미생활로 영상 편집하는 것을 하기 위해서 영상편집에 최적화 된 컴퓨터를 직접 조립해 보고 싶어지게 되었습니다.

컴퓨터 각 부품 기능, 가격, 선정이유


CPU :AMD 라이젠 7 2700X
기능 : 에너지와 소비 온도를 자체적으로 모니터링, 프로세서가 발열과 소음이 낮다고 판단되면 모든 응용프로그램에서 정확히 25MHz 단계로 클럭 속도를 높일 수 있음.
가격 :347,980원
선정 이유: : 가격이 싼 편은 아니지만 같은 가격대의 다른 CPU와 비교했을 때 굉장히 높은 효율성을 지니고 있음. 멀티 캐스팅이 뛰어남.


그래픽 카드 :이엠텍 XENON 지포스 RTX 2060 STORM X Dual OC D6 6GB
기능 : 게임에서의 실시간 레이 트레이싱, 고급 음영 기술,최신의 향상된 AI를 사용하는 그래픽, 그래픽카드 제어 및 모니터링 유틸리티
가격 :464,000원
선정 이유 : 3D효과나 작업이나 시네마코드를 짤때 그래픽 카드의 효과를 높게 볼수 있기 때문이다


메인보드 :에이수스 PRIME X470-PRO
기능 : 향상된 오디오, 강력한 전송속도, 최대 온도를 20도까지 낮추는 매우 효율적인 방열판,Aura 기능이 제공하는 다양한 조명효과를 활용하여 주파수 및 방향까지 조정 가능
가격 :208,000
선정 이유: : 제일 높은 등급의 칩셋이여서 고성능이다, 안정성과 내구성이 뛰어나다.


저장장치 : 마이크론 Crucial MX500 SSD 500GB
기능 : 설치 즉시 부팅속도에서 부터 작업,게임등에서의 성능 향상, 오랜 시간 사용해도 처음 같은 성능, 최대 2TB의 용량 제공
가격 :80,920원
선정 이유: 발열 및 소음 적고, 저장속도가 빠르다, 가성비가 좋다.


RAM : 삼성전자 DDR4 32G PC4-21300(16x2)
기능 : 고성능 메모리로 고용량 사진작업 및 고화질 영상작업, 렌더링, 인코딩 작업을 할때 오류를 최소화 함,낮은 소비전력
가격 :107,720x2원
선정 이유: 내구도가 좋고,고사양 작업에도 끊김이 없음,64Bit의 빠른 속도, 인코딩 오류 최소화


파워 : 시소닉 S12II-620 80 PLUS 브론즈
기능 : 과전압,저전압,단락,과전력보호회로를 탑재, 변환과정에서 필수적으로 발생하는 전기손실을 최소화
가격 :71,500원
선정 이유: 높은 내구성과 뛰어난 안정성,최대 87%의 뛰어난 전력 효율


케이스 : 3RSYS L530 화이트 강화유리
기능 : 상단에 I/0 PORT를 구성하여 확장성과 편리성을 높임, 튜닝에 확실한 측면 Full 강화유리를 통해 견고함과 탁트인 시야를 제공
가격 :68,000원
선정 이유: 소음이 적음, 발열이 잘 안됨, 쿨링팬 5개 내장

조립 후기

케이스의 크기는 상당히 큰편이고,내부 부품들이 깔끔하게 정리되어 외관상의 분위기도 괜찮다.
영상편집과 디자인 작업을 하는데 있어서 충분한 사양을 제공 할 수 있고, 성능은 4k 영상편집까지 가능할 수준이다.
CPU나 메모리용량, 그래픽카드까지 딱 알맞게 갖춰진 최적의 시스템이다.
